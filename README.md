# TALA v2 — نظام القيود المحاسبية الذكي

## التشغيل المحلي (localhost)

```bash
pip install -r requirements.txt
python app.py
```
افتح: http://localhost:5000

## المميزات الجديدة في v2
- واجهة احترافية مُعاد تصميمها بالكامل
- رسم بياني للتدفق النقدي (Cash Flow) على الداشبورد
- استيراد/تصدير CSV لكل الجداول (حسابات، موردين، عملاء، مصروفات، مواقع)
- إجراءات تعديل + حذف لكل السجلات عبر نوافذ (Modals)
- فصل كود الحساب عن اسمه في جميع الشاشات
- مديول البيانات الموحدة بهيكل جدول واضح (نوع / اسم النظام / اسم البنك / كود الحساب / اسم الحساب)
- المطابقة وعرض القيود مباشرة بعد رفع الكشف
- إصلاح توليد القيود: رسوم البنك وضريبتها منفصلتان عن مبلغ المورد

## الملفات
- `app.py` — Flask backend + جميع API endpoints
- `templates/index.html` — الواجهة الكاملة
- `matcher.py` — محرك المطابقة
- `parser.py` — تحليل كشوف البنوك
- `exporter.py` — تصدير QuickBooks CSV + Excel
